
# 🧠 SmartQuizzer – Adaptive AI-Based Quiz Generator

An AI-powered adaptive quiz application built using Streamlit and Python.

## 🚀 Features
- Upload study material (TXT/PDF)
- Adaptive quiz (Easy, Medium, Difficult)
- Timer-based questions
- Scoreboard and performance summary
- AI-based explanations (optional)

## 🛠 Tech Stack
- Python
- Streamlit
- OpenAI API
- NLP concepts

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
